의생명데이터분석 프로젝트(일단하조)
=============
OD600데이터 분석 및 시각화 툴
-------------

> 차의과학대학교 의생명데이터분석 수업의 팀 프로젝트 과제입니다.
> 3인 모두 파이썬 입문자이며 해당 프로젝트는 미생물의 OD(Optical Density) 데이터를 분석하여 성장 속도와 세대 시간을 통해 성장 곡선을 그려주는 시각화 툴을 만들어 보는 것입니다.
> 프로젝트 담당자는 각각 장영준-역할, 김수민-역할, 이지영-역할입니다.

> 사용기술
> > - Python 3.11  
> > - pandas, numpy, matplotlib, scipy  
> > - GitHub (버전관리)  
> > - Kakao talk, Figma (아이디어·디자인 논의)  
> > - Canva (발표 자료 제작)  
> > - Google Spreadsheet와 Local File (데이터 관리)
